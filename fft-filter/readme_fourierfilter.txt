
fourierfilter.maxpat and fourierfiltertest.maxpat

fourierfiltertest is the main patch which you should load. The pfft~ object within it will load an instance of fourierfilter automatically. Doubleclick on the pfft~ object to see fourierfilter at work.

Loading an independant instance of fourierfilter is only useful for editing.
